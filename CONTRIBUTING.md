# Contributing

This is an Open Business Blueprint + actor repo (maturity: **implemented**).
Contributions are welcome: jurisdiction/compliance-catalog research
(`src/knitwear/facts.cljc`, always cite an official spec-basis — never
fabricate a jurisdiction's requirements), governor HARD-check refinements
(`src/knitwear/governor.cljc`), and store/advisor extensions. Keep all
proposal ops inside the closed allowlist in `knitwear.governor` /
`knitwear.registry` — adding a new op requires updating both plus tests.
Open an issue or PR. License: AGPL-3.0-or-later.
